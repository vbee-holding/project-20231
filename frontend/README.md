## Getting Started

First:
```bash
npm i
```


run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

### Config

Create a .env file in the frontend folder and add the following data

```bash
GOOGLE_CLIENT_ID=
```
